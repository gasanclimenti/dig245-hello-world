  5. Learn-Computing:
    • File Organization & Naming Conventions: This learning-computing lesson explains the importance of clearly-labeled files. Organizing and structuring files can feel tedious but pays off as you return to the files in the future. Using full descriptions and dates in the names of your files can help for easy access to the exact file you are looking for. File names should not include spaces and be written in all lowercase, or another common naming convention.
    • The Filesystem: This learn-computing lesson talked about files as organizers of data, which are then organized into folders, which are organized into programs. Users can have access to these files through permissions, which include options to read (r), write (w), and/or execute (x) the file. Files are organized into heirarchical systems, which are contained by the root and can be located through paths. Certain files include extensions to identify the type of file they are. Additionally, files can be hidden.
  6. Web-Based Projects (selected from the spreadsheet list):
    • How Generative Music Works: This project described how music was generated digitally through different software. Each piece of music is created by different systems, which were created by the music-makers (the system, not the musician, creates the songs).
